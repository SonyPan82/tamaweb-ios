# Tamaweb iOS

iOS port of **Tamaweb** as an iPhone application, with native iOS integration built around the original project.

This repository is an iOS adaptation of the original web project; the goal is to offer a real iPhone version with native integration.

## Credits

All credit for the original game goes to its creator and source project:

- Original project: **Tamaweb**
- Creator / original repository: `https://github.com/autosam/Tamaweb`

This repository is solely a porting, integration, and iOS adaptation work.

## What this port includes

- Capacitor integration
- Xcode iOS project
- Interface adaptation for iPhone
- Mobile and retina display adjustments
- Native iOS integration for certain port features
- Native iPhone notifications tied to the Tamagotchi and garden state

## Availability

The app will be available for free on the App Store under the name Tamaweb iOS.

## Important

If you are looking for the original project, refer to the Tamaweb source repository.

If you are looking for the iPhone version of this port, this repository documents the iOS adaptation.
